## 🌐 طرق النشر السريعة

### الطريقة الأولى: GitHub Pages (الأفضل)

```bash
# 1. فتح Command Prompt وانتقل للمجلد
cd "c:\Users\DELL\OneDrive\Desktop\غخ\D.ST"

# 2. إنشاء مستودع git
git init

# 3. إضافة جميع الملفات
git add .

# 4. حفظ التغييرات
git commit -m "Initial commit"

# 5. تثبيت الفرع الرئيسي
git branch -M main

# 6. إضافة الرابط الأساسي (استبدل username)
git remote add origin https://github.com/your-username/your-username.github.io.git

# 7. رفع الملفات
git push -u origin main
```

**النتيجة**: موقعك سيكون متاح على:
```
https://your-username.github.io
```

## ⚡ الطريقة السريعة (Netlify)

### 1️⃣ أنشئ حساب GitHub (إذا لم تكن لديك)

- اذهب إلى https://github.com
- اختر Sign up
- ملء البيانات وأنهِ

### 2️⃣ ارفع الملفات

```bash
cd "c:\Users\DELL\OneDrive\Desktop\غخ\D.ST"
git init
git add .
git commit -m "موقع الملخصات والتمارين"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
git push -u origin main
```

### 3️⃣ انشر على Netlify

1. اذهب إلى https://netlify.com
2. اختر "New site from Git"
3. ربط GitHub
4. اختر المستودع
5. اضغط Deploy!

### ✅ تم!

موقعك سيكون متاح على:
```
https://your-site-name.netlify.app
```

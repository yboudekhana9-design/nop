# 🧪 اختبار الموقع محلياً

## قبل النشر على الإنترنت، اختبر الموقع محلياً!

---

## الطريقة 1️⃣: استخدام Python (الأسهل)

### الخطوة 1: فتح Command Prompt

اضغط على Windows Key وابحث عن `Command Prompt`

### الخطوة 2: الانتقال للمجلد

```bash
cd "c:\Users\DELL\OneDrive\Desktop\غخ\D.ST"
```

### الخطوة 3: تشغيل السيرفر

```bash
python -m http.server 8000
```

### الخطوة 4: فتح الموقع

افتح المتصفح وزيارة:
```
http://localhost:8000
```

### الخطوة 5: إيقاف السيرفر

اضغط `Ctrl + C` في Command Prompt

---

## 🔍 فتح أدوات المطور

لرؤية أي أخطاء:

1. اضغط `F12` أو `Ctrl + Shift + I`
2. اذهب إلى "Console"
3. تحقق من وجود رسائل خطأ حمراء

---

## 📱 اختبار على الهاتف المحمول

### من نفس الشبكة:

1. في Command Prompt، ابحث عن عنوان IP الخاص بك:
```bash
ipconfig
```

2. ابحث عن "IPv4 Address" (مثل: 192.168.1.100)

3. من الهاتف، افتح المتصفح وأدخل:
```
http://192.168.1.100:8000
```

موردة تعليمية

1️⃣ GitHub Pages (الأفضل والمجاني)

المميزات: مجاني، موثوق، دعم كامل

  1. أنشئ حساب على github.com
  2. أنشئ مستودع باسم: your-username.github.io
  3. رفع جميع الملفات للمستودع
  4. الموقع سيكون متاح على: https://your-username.github.io

الأوامر:

git init git add . git commit -m "Initial commit" git branch -M main git remote add origin https://github.com/your-username/your-username.github.io.git git push -u origin main

2️⃣ Netlify (الأسهل)

المميزات: سهل جداً، نشر تلقائي، واجهة بديهية

  1. زيارة netlify.com
  2. اختر "New site from Git"
  3. ربط حساب GitHub
  4. اختر المستودع الخاص بك
  5. اضغط Deploy! 🎉

الرابط: https://your-site-name.netlify.app

3️⃣ Vercel (السريع)

المميزات: سريع جداً، أداء ممتاز، دعم عالمي

  1. زيارة vercel.com
  2. اختر "New Project"
  3. استيراد من GitHub
  4. اختر المستودع
  5. اضغط Deploy! 🚀

الرابط: https://your-project.vercel.app

4️⃣ اختبار محلي (للتطوير)

المميزات: اختبر الموقع قبل النشر

  1. فتح Command Prompt
  2. الانتقال للمجلد
  3. تشغيل السيرفر المحلي
  4. زيارة http://localhost:8000

الأمر:

cd "c:\Users\DELL\OneDrive\Desktop\غخ\D.ST" python -m http.server 8000

📋 قائمة التحقق قبل النشر:

  • جميع الروابط تعمل بشكل صحيح
  • اللوغو يظهر بشكل صحيح
  • الموقع يبدو جيداً على الهاتف
  • لا توجد أخطاء JavaScript في Console
  • صفحات التحميل سريعة
  • جميع الصور تحميل بشكل صحيح
  • الألوان والتصميم متسقة
  • الملفات الأساسية موجودة جميعها

## 🎨 التصميم

### الألوان الأساسية
- **اللون الأرجواني:** `#667eea` - اللون الأساسي
- **اللون الأرجواني الداكن:** `#764ba2` - التدرج الثاني
- **الأبيض:** `#ffffff` - الخلفية
- **الرمادي:** `#666666` - النصوص الثانوية

### الخطوط
- العائلة: Arial, Droid Arabic Kufi, Sans-serif
- اتجاه النص: RTL (من اليمين إلى اليسار)

### التأثيرات
- Gradient الخطي للهيدر والأزرار
- ظلال ناعمة للبطاقات
- تأثيرات التحويم (Hover Effects)
- رسوم متحركة سلسة

## 📄 الصفحات الرئيسية

### 1. الصفحة الرئيسية (index.html)
- صورة بطل (Hero Image)
- شبكة بطاقات المستويات التعليمية
- روابط سريعة لكل مستوى
- معلومات عن مميزات الموقع

### 2. التعليم الابتدائي (primary.html)
- تقسيم حسب المستويات (تحضيري، 1AP، 2AP، إلخ)
- نماذج من الدروس والتمارين
- جدول ملخص بعدد الموارد
- روابط لصفحات التمارين التفصيلية

### 3. التعليم المتوسط (middle.html)
- 4 مستويات (السنة 1 إلى 4 متوسط)
- 8 مواد في كل مستوى
- نماذج الفروض والامتحانات
- جداول شاملة للموارد

### 4. التعليم الثانوي (secondary.html)
- 3 مستويات (السنة 1 إلى 3 ثانوي)
- محتوى متقدم ومتخصص
- نماذج البكالوريا
- استراتيجيات النجاح

### 5. التمارين الشاملة (exercises.html)
- تمارين حسب المواد
- ملخصات سريعة قابلة للتحميل
- إحصائيات التمارين

### 6. صفحات إضافية
- **about.html** - معلومات عن الموقع والفريق
- **contact.html** - نموذج الاتصال
- **privacy.html** - سياسة الخصوصية
- **terms.html** - الشروط والأحكام

## 💻 التكنولوجيا المستخدمة

- **HTML5** - هيكل الصفحة
- **CSS3** - التصميم والتنسيق
  - Flexbox للتخطيطات
  - Grid للشبكات
  - Gradients للألوان
  - Media Queries للتجاوب
- **JavaScript** - التفاعلات البسيطة

  - معالجة النماذج
  - الرسوم المتحركة
